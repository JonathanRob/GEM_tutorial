Tutorial on Genome-Scale Metabolic Models (GEMs)

Overview

This tutorial consists of a collection of exercises aimed at providing an introduction to the basics of genome-scale metabolic model (GEM) structure and application.

  • Background
  • Exercise 1: GEM basics
  • Exercise 2: Flux balance analysis
  • Exercise 3: GEM extraction
  • Exercise 4: Comparison of extracted GEMs
  • Exercise 5: Reporter metabolites and subsystem enrichment

Software

  • MATLAB (R2018 or newer) (30-day trial)

Additional dependencies specific to each exercise will be stated at the start of each exercise.
